## Clone Repo

To get a local copy up and running follow these simple steps:

   - `Open terminal Ctrl + Alt + T`

   - `git clone git@github.com:francisuloko/chuck-norris-jokes.git`

   - `cd check-norris-jokes`

   - `npm install`

   - `npm start`

   - `visit localhost:8080`
